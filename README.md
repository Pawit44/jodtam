# ตัวช่วยอ่าน — Line-by-Line Reading Assistant

> คำอธิบายเว็บและวิธีใช้งาน (ไทย / อังกฤษ)
> App description and usage guide (Thai / English)

---

เว็บนี้คือตัวช่วยอ่านข้อความยาวๆ ทีละท่อนสั้นๆ เพื่อให้อ่านหรือจดตามได้ง่ายขึ้น

**วิธีใช้งาน**

1. ลบข้อความตัวอย่างออก แล้ววางข้อความหรือบทความที่ต้องการอ่านลงในช่องนี้แทน
2. กดปุ่ม **"เริ่มอ่าน"** ระบบจะแบ่งเนื้อหาออกเป็นท่อนสั้นๆ อ่านง่ายให้โดยอัตโนมัติ ไม่ตัดคำกลางคำ
3. ระหว่างอ่าน แตะที่ข้อความหรือกดปุ่ม **"ถัดไป"** เพื่อไปยังท่อนถัดไป หรือปัดย้อนกลับเพื่อดูท่อนก่อนหน้า
4. บนคอมพิวเตอร์สามารถกด `Space`, `Enter` หรือลูกศรบนคีย์บอร์ดแทนการคลิกเมาส์ได้เช่นกัน
5. เมื่ออ่านจบต้องการเริ่มใหม่ กดปุ่ม **"แก้ไข / วางข้อความใหม่"** ด้านล่างของหน้าอ่านได้ทุกเมื่อ

**พัฒนาโดย** Ur. | Mr.Atsawin Niranam — ภาควิชาวิศวกรรมคอมพิวเตอร์

---

This website is a reading assistant that breaks long text into short, easy-to-follow chunks, making it easier to read or take dictation from.

**How to use**

1. Delete the sample text, then paste the text or article you want to read into this box instead.
2. Press the **"Start Reading"** button. The system will automatically split the content into short, readable chunks — without cutting words in the middle.
3. While reading, tap the text or press the **"Next"** button to move to the next chunk, or swipe back to review the previous one.
4. On a computer, you can also press `Space`, `Enter`, or the arrow keys on your keyboard instead of clicking with the mouse.
5. When you're done reading and want to start over, press the **"Edit / Paste New Text"** button at the bottom of the reading screen at any time.
